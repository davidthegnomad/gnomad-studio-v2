# McGaugh Ranch Website

This folder contains the complete source code and build for the McGaugh Ranch website.

## Critical Requirement
This is a **Vite + React** project. You **cannot** open `index.html` directly in a browser or serve it via a simple file server (like Python's `http.server` or Apache) unless you are serving the built files in the `dist/` directory.

## Deployment Options

### 1. Simple Hosting (Recommended)
If you are using a static host (Netlify, Vercel, Firebase Hosting, etc.):
- Upload the **contents** of the `dist/` folder.
- The `dist/` folder is generated by running `npm run build`.

### 2. VPS or Node.js Server
If you are running this on your own Linux server:
1. Upload all files from this `upload_server` directory.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a production build:
   ```bash
   npm run build
   ```
4. Serve the `dist/` folder using Nginx, Apache, or Vite's preview:
   ```bash
   npm run preview -- --host
   ```

## Local Development
To run the project locally for development:
```bash
npm install
npm run dev
```

## Configuration
Set your `GEMINI_API_KEY` in a `.env.local` file:
```bash
GEMINI_API_KEY=your_actual_key_here
```

## Troubleshooting
- **MIME Type Error**: If you see a "MIME type" error, it means you are likely trying to serve `.tsx` files directly without the Vite development server. Use `npm run dev`.
- **Tailwind CDN**: The project has been upgraded to use a proper Tailwind PostCSS build for production performance and compliance.
